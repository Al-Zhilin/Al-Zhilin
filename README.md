## Ключевые проекты

### [PoolControl](https://github.com/Al-Zhilin/PoolControl)
`ESP32` `C++` `FreeRTOS` `VK Bot API` `GitHub Actions (OTA)`

> Система автоматического контроля нагрева: многозадачность на FreeRTOS, панель мониторинга и управления в VK (Bots Long Poll API, обновление интерфейса в реальном времени), автоматическое обновление прошивки через GitHub Actions.

---

### [OnlineScales](https://github.com/Al-Zhilin/OnlineScales)
`ESP32` `HX711` `SIM800L (GSM)` `Yandex Cloud Functions`

> Система дистанционного весового контроля на базе тензодатчиков: компенсация температурного дрейфа, отправка данных через GSM-модуль, обработка данных в облаке, настраиваемые уведомления и конструирование BI-аналитики.

---

### [Signalization](https://github.com/Al-Zhilin/Signalization)
`ESP32` `Telegram Bot API` `DS18B20`

> Система охранной сигнализации: удалённое управление через Telegram (long polling), контроль пожарных датчиков и датчиков движения.

---

### [Student_Attendance](https://github.com/Al-Zhilin/Student_Attendance)
`ESP32-S3` `Google Sheets API` `Telegram Bot API`

> Система электронного учёта посещаемости студентов: ведение журнала через API Google Sheets, удобное управление и полный функционал в интерфейсе чата.
